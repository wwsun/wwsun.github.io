这个仓库是 Anthropic 官方维护的 **Agent Skills 示例集合**，每个 skill 是一个独立文件夹，包含 `SKILL.md` 指令和配套脚本。完整的开发和构建指南可参考 [[building-skills-for-claude|构建 Claude 技能完整指南]]。按用途分为四大类：

---

### 🎨 创意与设计类

| #   | Skill                 | 作用                                                                                                                            |
| --- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| 1   | **algorithmic-art**   | 用 p5.js 创作**算法艺术/生成艺术**。支持流场、粒子系统、种子随机性等。输出 `.html`（交互式）和 `.js`（生成算法）。              |
| 2   | **canvas-design**     | 用设计哲学创作**海报、视觉作品**，输出 `.png`/`.pdf`。分两步：先创建设计哲学（`.md`），再将其视觉化表达。                       |
| 3   | **frontend-design**   | 创建**生产级前端界面**：网站、落地页、Dashboard、React 组件、HTML/CSS 布局。强调设计质量，避免"AI 味"审美。                     |
| 4   | **slack-gif-creator** | 制作**Slack 专用 GIF 动图**（Emoji 128×128，消息 480×480）。提供 GIFBuilder、缓动函数、验证工具和动画概念（弹跳/脉冲/旋转等）。 |
| 5   | **theme-factory**     | 为文档、幻灯片、HTML 页面等**套用主题样式**。内置 10 套预设主题（配色+字体），也可即时生成新主题。                              |

### 💻 开发与技术类

| #   | Skill                     | 作用                                                                                                                                                        |
| --- | ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 6   | **claude-api**            | **Claude API/Anthropic SDK 开发全流程**：构建、调试、优化，包含 Prompt Caching、模型版本迁移（4.5→4.6→4.7）、Thinking、Tool Use、Batch 等所有 Claude 特性。 |
| 7   | **mcp-builder**           | 指导创建**高质量 MCP（Model Context Protocol）服务器**。支持 Python（FastMCP）和 Node/TypeScript（MCP SDK），让 LLM 通过工具与外部服务交互。                |
| 8   | **webapp-testing**        | 用 **Playwright** 测试本地 Web 应用。验证前端功能、调试 UI 行为、截图、查看浏览器日志，支持管理多服务生命周期。                                             |
| 9   | **web-artifacts-builder** | 用 **React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui** 构建复杂、多组件的 Claude.ai Artifact。支持状态管理和路由，最终打包成单个 HTML 文件。        |

### 🏢 企业与沟通类

| #   | Skill                | 作用                                                                                                                        |
| --- | -------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| 10  | **brand-guidelines** | 将 **Anthropic 官方品牌色和字体**应用到任何输出物中，确保视觉一致性。                                                       |
| 11  | **doc-coauthoring**  | 引导用户完成**结构化文档协作写作流程**。三阶段：上下文收集 → 精炼与结构化 → 读者测试。适用 PRD、设计文档、RFC、技术规格等。 |

### 📌 关键信息

- **许可证**：大多数 skill 为 **Apache 2.0** 开源；docx/pdf/pptx/xlsx 四个文档 skill 为 **Source Available**（可查看源码但非开源），它们是驱动 Claude 文档能力的生产级代码
- **使用方式**：在 Claude Code 中通过 `/plugin marketplace add anthropics/skills` 注册市场，然后安装；在 Claude.ai 付费版中这些 skill 已全部内置；在 API 中可通过 Skills API 上传自定义 skill
- **与 Agent Skills 标准的关系**：此仓库是 Anthropic 对该标准的实现参考，标准本身在 [agentskills.io](http://agentskills.io)
