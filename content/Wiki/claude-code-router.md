## install

```bash
npm install -g @anthropic-ai/claude-code

npm install -g @musistudio/claude-code-router
```

## Commands

```bash
# Start Claude Code using the router:
ccr code

# Restart Claude Code 
ccr restart

# UI Mode
ccr ui
```

## Config

`~/.claude-code-router/config.json`
