## 搜索技巧

### 使用双引号精确搜索

`"your question goes here"`

### 搜索指定站点：

`site:freecodecamp.org react`

### 从结果中排除 `-`

`how to write components in React -class`

### 搜索指定图片尺寸 `imagesize:`

`cute dog images imagesize:500x600`

### 搜索指定文件类型

`filetype:pdf`

### 模糊匹配

`the * of foo`

### 组合搜索

`OR, AND`
`angular and react`

### 指定日期范围

```
AFTER:2020
BEFORE: 2016
2020..2024

React tutorial AFTER:2020
```

### 搜索相似网站

`related:google.com`
